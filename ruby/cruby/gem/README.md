# NanoGL for cruby

NanoGL for cruby.

## Build

Build `libnanogl.a` beforehand.

In this folder, enter the following command.

```shell
$ make all
$ rake build
```

## Installation

In this folder, enter the following command.

```shell
$ gem install -l pkg/nanogl-<version>.gem
```

## Usage

Please see *.rb file in the example folder.
